# Write a Script that Compares Numbers

<h6> Key Concepts<h6>

* man test : **check file types and compare values**

* -eq       : **is equal**

* -ge       : **is greater than or equal to**

* -gt       :  **is greater than**

* -le       : **is  less  than or equal to**

* -lt       : **is less than**

* -ne       :  **is not equal to**
